# Теоретическая часть

## Генетический алгоритм

Генетический алгоритм (ГА) — это эвристический метод глобальной оптимизации, основанный на принципах
естественного отбора и генетической наследственности.

## PSO

Алгоритм роя частиц (PSO) — это метода оптимизации, основанный на наблюдениях за поведением групп
животных, таких как стаи птиц или косяки рыб.
