# Описание проекта (репозиторий algsi)

## 1. Назначение проекта
Проект предназначен для демонстрации работы эвристических методов оптимизации при поиске **локального минимума** целевых функций в 1D и 2D постановках. Реализованы и визуализируются два алгоритма:
- **Генетический алгоритм (GA)** — с модификацией в виде *арифметического кроссовера* и *элитизма*.
- **Particle Swarm Optimization (PSO)** — с модификацией в виде *ограничения скорости* (velocity clamping) и *линейного уменьшения* `vmax` по итерациям.

Ключевая особенность проекта — наличие графического интерфейса (Tkinter) и визуализация процесса сходимости.

## 3. Основной модуль `main.py`

### 3.1. Используемые библиотеки
Таблица 2. Импортируемые библиотеки `main.py`

| Библиотека | Назначение |
|---|---|
| `tkinter`, `tkinter.ttk` | Графический интерфейс приложения. |
| `numpy` | Векторные вычисления, генерация случайных величин, вычисление целевых функций. |
| `matplotlib` + `FigureCanvasTkAgg` | Визуализация: графики функций, точки популяции/роя, история сходимости. |

### 3.2. Целевые функции
Таблица 3. Целевые функции

| Функция | Область поиска | Описание |
|---|---:|---|
| `f(x)` | `x ∈ [0, 20]` | 1D функция: `f(x) = x·sin(x)` |
| `f2(x, y)` | `x, y ∈ [-10, 10]` | 2D многомодальная функция (по коду — с `sin`, `exp` и степенью `0.1`). |

### 3.3. Вспомогательная функция снимков
Таблица 4. Глобальные служебные функции

| Функция | Публичность | Описание |
|---|---|---|
| `_snapshot_indices(total, max_snapshots=8)` | Приватная (служебная) | Возвращает индексы поколений/итераций для сохранения ограниченного числа «снимков» популяции/роя (для визуализации). |

### 3.4. Генетический алгоритм (GA)
Реализованы две функции: для 1D и для 2D. В обеих версиях оптимизация — **минимизация**.

Таблица 5. GA (1D)

| Функция | Публичность | Описание |
|---|---|---|
| `genetic_algorithm(pop_size, generations, crossover_prob, mutation_prob, mutation_scale)` | Публичная | Минимизирует `f(x)` на `[0,20]`. Реализует турнирный отбор (размер 3), элитизм, арифметический кроссовер и мутацию гауссовским шумом. Возвращает лучший `x`, значение `f(x)`, историю лучших значений и снимки популяции. |

Таблица 6. GA (2D)

| Функция | Публичность | Описание |
|---|---|---|
| `genetic_algorithm_2d(pop_size, generations, crossover_prob, mutation_prob, mutation_scale)` | Публичная | Минимизирует `f2(x,y)` на `[-10,10]×[-10,10]`. Аналогично 1D: турнир, элитизм, арифметический кроссовер по координатам и мутация гауссовским шумом. Возвращает лучший `(x,y)`, значение, историю и снимки популяции. |

### 3.5. Роевой алгоритм (PSO)
Реализованы две функции: для 1D и для 2D. В обеих версиях оптимизация — **минимизация**.

Таблица 7. PSO (1D)

| Функция | Публичность | Описание |
|---|---|---|
| `pso(swarm_size, iterations, c1, c2, w, vmax)` | Публичная | Минимизирует `f(x)` на `[0,20]`. Использует `pbest`/`gbest`, ограничение скорости `vmax` и линейное уменьшение `vmax` по итерациям (не ниже 5% от исходного). Возвращает лучший `x`, значение, историю и снимки позиций. |

Таблица 8. PSO (2D)

| Функция | Публичность | Описание |
|---|---|---|
| `pso_2d(swarm_size, iterations, c1, c2, w, vmax)` | Публичная | Минимизирует `f2(x,y)` на `[-10,10]×[-10,10]`. Аналогично 1D, но скорости и позиции ведутся раздельно для `x` и `y`. Возвращает лучший `(x,y)`, значение, историю и снимки позиций. |

## 4. Графический интерфейс (GUI)
GUI реализован двумя классами: главное окно и окно параметров.

### 4.1. Класс `App`
Таблица 9. Методы класса `App`

| Метод | Публичность | Описание |
|---|---|---|
| `__init__()` | Публичный | Создаёт главное окно приложения и вызывает сборку интерфейса. |
| `_build_main()` | Приватный (логически) | Формирует главное окно: выбор целевой функции (1D/2D) и метода (GA/PSO). |
| `_open_ga()` | Приватный (логически) | Открывает окно параметров для GA (`ParamWindow`). |
| `_open_pso()` | Приватный (логически) | Открывает окно параметров для PSO (`ParamWindow`). |

### 4.2. Класс `ParamWindow`
Таблица 10. Атрибуты (параметры по умолчанию)

| Атрибут | Публичность | Описание |
|---|---|---|
| `GA_PARAMS` | Публичный (как константа) | Набор полей ввода для GA (размер популяции, поколения, вероятности, масштаб мутации). |
| `PSO_PARAMS` | Публичный (как константа) | Набор полей ввода для PSO (размер роя, итерации, `c1`, `c2`, `w`, `vmax`). |

Таблица 11. Методы класса `ParamWindow`

| Метод | Публичность | Описание |
|---|---|---|
| `__init__(parent, method, func="1d")` | Публичный | Создаёт окно, сохраняет выбранные метод и тип функции, инициализирует UI. |
| `_build()` | Приватный (логически) | Рисует поля параметров, кнопку запуска и область под графики. |
| `_get(key, cast=float)` | Приватный (логически) | Считывает значение из поля ввода и приводит тип. |
| `_run()` | Приватный (логически) | Запускает выбранный алгоритм (GA/PSO, 1D/2D), отображает численный результат и вызывает отрисовку графиков. |
| `_draw(best_x, best_y, history, snapshots)` | Приватный (логически) | Визуализация 1D: график функции, точки популяции/роя (снимки) и график сходимости. |
| `_draw_2d(best_x, best_y, best_val, history, snapshots)` | Приватный (логически) | Визуализация 2D: контурная карта, точки популяции/роя и график сходимости. |

## 6. Запуск
Стандартный запуск приложения:

```bash
python main.py
```

Для работы необходимы:
- Python 3.x
- `numpy`
- `matplotlib`
- `tkinter` (обычно установлен вместе с Python, но зависит от ОС)
