# Описание проекта

## Введение
Данный проект направлен на реализацию и сравнение различных алгоритмов, включая базовые алгоритмы и алгоритмы на основе генетических решений.

## Модули проекта

| Модуль                      | Описание                                                 |
|-----------------------------|----------------------------------------------------------|
| `main.py`                  | Главный модуль, запускающий алгоритмы.                   |
| `baseline_algorithms.py`   | Реализация базовых алгоритмов сортировки и оптимизации.  |
| `compare_algorithms.py`    | Модуль для сравнения алгоритмов, в данный момент имеет неподходящие импорты. |
| `README.md`                | Описание проекта и его использование.                     |
| `BLOCK_DIAGRAM_MAIN.mmd`   | Блок-схема, описывающая структуру проекта.              |
| `PROGRAM_DESCRIPTION.md`    | Подробное описание программы и её функциональности.      |
| `assets.pdf`               | Ассеты и документация проекта в формате PDF.            |
| `.gitignore`               | Список файлов и каталогов, которые не должны отслеживаться Git. |

## Описание `compare_algorithms.py`
Модуль `compare_algorithms.py` предназначен для сравнения различных алгоритмов, однако в текущем состоянии он содержит неверные импорты. Конкретно:
- Неподходящие импорты `genetic_algorithm.py`, `genetic_algorithm_2d.py` и неверные названия `baseline_ga/baseline_pso`. Эти импорты следует исправить, чтобы модуль корректно работал с существующими файлами и функциями.

## Заключение
Проект содержит важные модули, которые реализуют алгоритмы и представляют результаты их работы. Важно исправить текущие ошибки в модулях для достижения корректности работы программы.
